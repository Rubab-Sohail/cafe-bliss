# Café Bliss ☕

A fully responsive restaurant website built with React and Vite. The project includes multiple pages, smooth navigation, a reservation form, and a contact form.


## 📸 Pages

- **Home** – Hero section, signature menu preview, ambience gallery, guest reviews
- **About** – Restaurant story, core values, team section, venue photos
- **Contact** – Contact details and working message form
- **Reservations** – Full table booking form with date, time, and guest selection

---

## 🛠️ Built With

- [React 18](https://react.dev/)
- [React Router v6](https://reactrouter.com/)
- [Vite](https://vitejs.dev/)
- CSS (custom, no UI library)

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Rubab-Sohail/cafe-bliss.git
cd cafe-bliss
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### 4. Build for production
```bash
npm run build
```
